
### 🧮 Calculadora de IMC 🚧

Aplicação em Python que calcula o Índice de Massa Corporal (IMC) com foco em lógica de programação, validação de dados e evolução para um sistema completo.

---

### 🧠 Objetivo do projeto

Evoluir de um exercício simples para uma aplicação mais robusta, aplicando conceitos fundamentais de programação e boas práticas.

---

### ✅ Funcionalidades já implementadas

**🧱 Passo 1 — Lógica simplificada**

* Uso de condições mais limpas (`< 18.5`, `< 25`, `< 30`, `else`)
* Código mais legível e com menos chance de erro

**🧪 Passo 2 — Validação de entrada**

* Verificação de valores inválidos (peso e altura ≤ 0)
* Prevenção de erros como divisão por zero

**🧠 Passo 3 — Saída melhorada**

* Exibição do IMC com 2 casas decimais
* Mensagens mais claras e informativas para o usuário

---

### 🚧 Em desenvolvimento (roadmap)

**💡 Passo 4 — Cálculo de peso ideal**

* Cálculo do peso mínimo e máximo com base no IMC saudável
* Uso da fórmula: `peso = IMC × altura²`

**🎯 Passo 5 — Recomendações personalizadas**

* Indicar quanto o usuário precisa ganhar ou perder peso
* Tornar o sistema mais útil e prático

**🔁 Passo 6 — Loop de execução**

* Permitir múltiplos cálculos sem reiniciar o programa

**⚙️ Passo 7 — Menu interativo**

* Estrutura com opções:

  * Calcular IMC
  * Ver explicação das faixas
  * Sair

**📊 Passo 8 — Histórico de dados**

* Armazenamento de:

  * Peso
  * Altura
  * IMC
* Exibição dos últimos cálculos

### ▶️ Como executar o projeto

1. Clone o repositório:

```
git clone https://github.com/gabriellechristie-dev/mini-projetos.git
```

2. Acesse a pasta do projeto:

```
cd mini-projetos/calculadora-imc
```

3. Execute o arquivo:

```
python calculadora_imc.py
```
